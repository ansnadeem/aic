Automatic Issue Classifier

Automatic Issue Classifier (AIC) is a Transfer Learning based application that automatically classifies issues on GitHub repositories

The application, once integrated with a repositories assigns a label to any newly created issue

The following default GitHub labels are currently supported.

  • Bug
  • Enhancement
  • Question

AIC leverages pretrained Transfer Learning based models and is fine-tuned on text extracted from real GitHub repositories

Installation Instructions

The application can be installed with a single click by heading to the app page https://github.com/apps/automatic-issue-classifier and clicking install, then selecting the repositories to integrate the application with.
